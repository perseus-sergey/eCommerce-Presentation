Эффективные результаты

На старте проекта мы сразу договорились о структуре нашего проекта, его тематике и дизайне.

На основе наших договоренностей мы использовали понятные и логичные методологии написания кода.

Например, в ходе наших совместных рассуждений мы пришли к соглашению использовать в нашем проекте только один слушатель событий кликов.

Для этого мы разработали специальный модуль, принимающий и обрабатывающий все клики.

Функция обработки кликов на основе делегирования событий:

              private clickHandler = (event: Event): void => {
                const element = event.target as HTMLElement;
                if (element.closest('a')) event.preventDefault();
                this.selectorHandlers.some((oSelectors) => {
                  oSelectors.selectors.some((oSelector) => {
                    if (element.closest(oSelector)) {
                      oSelectors.handler(event);
                      return true;
                    }
                    return false;
                  });
                  return false;
                });
              };

Поэтому наш проект не перегружен зависшими и не используемыми click Listener-ами.

Использованные технологии

В своей работе мы использовали Type script и современные методологии ES-6 спецификации Java Script (Классы, модульную систему, Map, стрелочные функции, деструктуризацию, промисы...)


              export const generateDomElement = (
                tag: string,
                innerHTML?: string | null | undefined,
                parent?: HTMLElement | null,
                ...classes: string[] | [string[]]
              ): T => {
                const element = document.createElement(tag);
                const arrClasses: string[] = [];
                let elementId = '';
              
                const fillArray = (selector: string): void => {
                  const [, separ, name] = selector.split(/([.#])/);
                  if (separ === '.') arrClasses.push(name);
                  else if (separ === '#') elementId = name;
                };
              
                classes.forEach((className) => {
                  if (Array.isArray(className)) {
                    className.forEach((classInside) => fillArray(classInside));
                  } else if (className) className.split(' ').forEach((splittedClass) => fillArray(splittedClass));
                });
                if (arrClasses.length) element.classList.add(...arrClasses);
                if (elementId) element.id = elementId;
                if (innerHTML) element.innerHTML = innerHTML;
                if (parent) parent.append(element);
                return element;
              };
