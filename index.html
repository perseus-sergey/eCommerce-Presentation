FOUT vs. FOUT with Class

FOUT

It's better off reading words written in system fonts than words written in invisible ink

FOUT with Class

Uses JavaScript to load the fonts

  1. Download the system fonts.
  2. Load web fonts via JavaScript.
  3. When loading web fonts, add a class to the html tag.
  4. Switch the system font to the downloaded web font.

Using FOUT with Class instead of @font-face + font-display: swap

  • We can group color changes.
  • We can adapt to user preferences.
  • We can skip downloading fonts entirely if users have a slow connection.

FOFT
(flash of faux text)

Split the font loading into three steps

  1. Use a fallback font if the web fonts are not already loaded.
  2. Download the Roman font first.
  3. Download the rest of the fonts.

Using Standard FOFT

First, load the "Roman font":

              @font-face {
                font-family: LatoInitial;
                src: url('path-to-lato.woff2') format('woff2'),
                     url('path-to-lato.woff') format('woff');
                unicode-range: U+65-90, U+97-122;
              }
              
              .fonts-loaded-1 body {
                font-family: LatoInitial;
              }
            

              if('fonts' in document) {
                document.fonts.load("1em LatoInitial")
                  .then(_ => {
                    document.documentElement.classList.add('fonts-loaded-1')
                  })
              }
Then we load other fonts

              /* Load this first */
              @font-face {
                font-family: LatoInitial;
                src: url('path-to-lato.woff2') format('woff2'),
                     url('path-to-lato.woff') format('woff');
                unicode-range: U+65-90, U+97-122;
              }
              
              /* Load these afterwards */
              @font-face {
                font-family: Lato;
                src: url('path-to-lato.woff2') format('woff2'),
                     url('path-to-lato.woff') format('woff');
                unicode-range: U+65-90, U+97-122;
              }
              
              /* Other @font-face for different weights and styles*/
              
              .fonts-loaded-1 body {
                font-family: LatoInitial;
              }
              
              .fonts-loaded-2 body {
                font-family: Lato;
              }
Add fonts-loaded-2 to the HTML

              function loadFonts () {
                if (sessionStorage.fontsLoaded) {
                  document.documentElement.classList.add('fonts-loaded-2')
                  return
                }
              
                if ('fonts' in document) {
                  document.fonts.load('1em Lato')
                    .then(_ => {
                      document.documentElement.classList.add('fonts-loaded-1')
              
                      Promise.all([
                        document.fonts.load('400 1em Lato'),
                        document.fonts.load('700 1em Lato'),
                        document.fonts.load('italic 1em Lato'),
                        document.fonts.load('italic 700 1em Lato')
                      ]).then(_ => {
                        document.documentElement.classList.add('fonts-loaded-2')
              
                        // Optimization for Repeat Views
                        sessionStorage.fontsLoaded = true
                      })
                    })
                }
              }

Critical FOFT

Load only the necessary CSS before loading everything else to improve performance.

Only the letters A to Z (uppercase and lowercase) are critical.

@font-face declaration looks like

              @font-face {
                font-family: LatoSubset;
                src: url('path-to-optimized-lato.woff2') format('woff2'),
                     url('path-to-optimized-lato.woff') format('woff');
                unicode-range: U+65-90, U+97-122;
              }
First we load this subset.

              .fonts-loaded-1 body {

              if('fonts' in document) {
                document.fonts.load('1em LatoSubset')
                  .then(_ => {
                    document.documentElement.classList.add('fonts-loaded-1')
                  })
              }
Load other font files later.

              .fonts-loaded-2 body {
                font-family: Lato;
              }
